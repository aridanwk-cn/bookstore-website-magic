# 书香阁 - 精品书籍展示网站

一个精美、响应式的书籍展示网站，专门用于推荐和展示各类优质书籍。

## 🌟 项目特色

### 页面设计
- **精美风格**：采用优雅的配色方案，绿色主题搭配金色点缀，营造书香氛围
- **清晰布局**：模块化设计，层次分明，信息展示清晰
- **响应式设计**：完美适配桌面端、平板和移动设备

### 核心功能
- **首页轮播**：引人入胜的欢迎页面，突出品牌特色
- **热门爆款推荐**：专门展示当前最受欢迎的畅销书籍
- **分类浏览**：按文学小说、科学技术、历史人文、经济管理等分类浏览
- **新书上架**：展示最新上市的优质书籍
- **智能搜索**：支持按书名、作者、描述、标签搜索
- **交互效果**：平滑滚动、悬停动画、动态加载等

## 🚀 快速开始

### 直接打开
1. 双击 `index.html` 文件即可在浏览器中打开
2. 或使用本地服务器运行以获得更好的体验

### 使用本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 🎨 设计特色

### 色彩方案
- **主色调**：深绿色 (#2c5530) - 代表知识的深度
- **辅助色**：金色 (#d4a017) - 象征智慧的光芒
- **背景色**：浅灰色 (#f7fafc) - 提供舒适的阅读体验

### 字体选择
- **标题字体**：Noto Serif SC - 优雅的衬线字体，适合中文
- **正文字体**：Inter - 现代无衬线字体，阅读舒适

### 交互效果
- **平滑滚动**：页面内导航平滑过渡
- **悬停动画**：卡片悬停时的微动效
- **加载动画**：页面加载时的淡入效果
- **返回顶部**：滚动时显示返回顶部按钮
